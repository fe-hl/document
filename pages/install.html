安装 #

环境支持 #

@fe-hl/admin-design-vue 可以在支持 ES2018 and ResizeObserver. 的浏览器上运行。 如果您确实需要支持旧版本的浏览器，请自行添加 Babel 和相应的 Polyfill .

由于 Vue 3 不再支持 IE11，@fe-hl/admin-design-vue 也不再支持 IE 浏览器。.

Edge ≥ 79Firefox ≥ 78Chrome ≥ 64Safari ≥ 12

Version #

@fe-hl/admin-design-vue 目前还处于快速开发迭代中。

使用包管理器 #

我们建议您使用包管理器 (NPM, Yarn, pnpm) 安装 Element Plus, 然后您就可以使用打包工具，例如 Vite and webpack.

shell
# Choose a package manager you like.

# NPM
$ npm install @fe-hl/admin-design-vue --save

# Yarn
$ yarn add @fe-hl/admin-design-vue

# pnpm
$ pnpm install @fe-hl/admin-design-vue

如果您的网络环境不好，建议使用相关镜像服务 cnpm 或 Alibaba.
